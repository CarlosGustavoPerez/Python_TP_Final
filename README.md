# DataPilot AI

## Descripción

Aplicación web para el análisis inteligente de datasets utilizando Pandas y modelos de lenguaje (LLMs).

## Funcionalidades

- Carga de archivos CSV
- Análisis exploratorio
- Calidad de datos
- Correlaciones
- Detección de outliers
- Insights automáticos
- Informe generado con IA
- Chat sobre el dataset

## Tecnologías

- Python
- Streamlit
- Pandas
- Plotly
- Hugging Face
